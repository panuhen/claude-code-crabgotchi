# Changelog

All notable changes to Claude Code Crabgotchi will be documented in this file.

## [0.1.0] - 2025-01-21

### Added

- Initial release
- 12 reactive emotions based on Claude Code activity
- Tamagotchi-style stats (Food, Happiness, Energy)
- Feed and Pet interactions with animations
- Break timer with Pomodoro-style notifications
- Customizable crab and bubble colors (7 options each)
- Night mode with stars and moon (11 PM - 5 AM)
- Easter eggs for Ferris and Claude mentions
- Real-time Claude Code log monitoring
- Pattern-based emotion triggers
